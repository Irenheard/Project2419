Project Abstract

Kawartha Conservation is a watershed-based conservation authority that was established in 1979. They manage the natural resources features within the 4800 square kilometers of land within their watershed boundary. Part of their responsibilities include monitoring the changes in the environment using the Ecological Land Classification (ELC) system by recording the ecosite units. The main objectives of this project were to use Kawartha Conservation’s previous Ecological Land Classification data to create deep learning data training sets, and train and assess an Esri deep learning model that could assist Kawartha Conservation with the Ecological Land Classification process. The team achieved this using the tools in ArcGIS Pro and found that the created deep learning model could not classify the ecosites in a way that would be beneficial to Kawartha Conservation. Despite these results, this project will still benefit Kawartha Conservation as it provided a valuable insight into the limits of image classification.

Background Information

Kawartha Conservation aims to monitor the changes in the land use throughout their watershed as the land use impacts aspects that they must monitor, such as water quality, quantity, holding capacity, and hydrology. Land use is also a factor in soil erosion and flood hazards, which Kawartha Conservation is mandated to regulate.

As of now, Kawartha Conservation determines and delineates the different ecosites through manual aerial imagery interpretation. This is a laborious and time-consuming process. As per the clients’ request, this project created a deep learning model and sorted out a workflow to automatically classify imagery and delineate the ELC boundaries, to help aid Kawartha Conservation with the ELC process.

What's Ecological Land Classification?

The Ecological Land Classification (ELC) system is a system used in Ontario designed to categorize the land into different classifications based on the natural features of the land, such as climate, bedrock type, vegetation, and soil type. There are six units to the ELC system:

  • Ecozones
  • Ecoregions
  • Ecodistricts
  • Ecosections
  • Ecosites
  • Ecoelements

Ecozones, Ecozones are the largest unit, while ecoelements are the smallest unit. The larger units are composed of multiple smaller units. For example, each ecozone will have multiple ecoregions within it, and each of those ecoregions will have multiple ecodistricts within it. This pattern continues down until the ecoelements, as there are not any smaller units. Ecosites are important as they are used by conservation authorities for planning and conservation efforts, among other things.

What's Deep Learning?

Deep learning is a subset of machine learning. Both technologies are capable of analyzing data and making decisions based off the data, however, deep learning is a little more complex than traditional machine learning. Deep learning analyzes data on a deeper level, using neural networks that have multiple layers to them. Each layer analyzes the data a little differently, looking for trends. Deep learning has a variety of applications within GIS, as it can be used for image classification, object detection, semantic segmentation, instance segmentation, panoptic segmentation, and image translation.
